# 📟 Morse Code

## 📌 Overview
Morse code is a classic communication system that represents letters, numbers, and punctuation using sequences of short and long signals: **dots (·)** and **dashes (–)**. 

Whether transmitted via sound, light flashes, or electrical pulses, Morse code stands as one of the earliest and most resilient forms of long-distance digital communication. It is globally recognized for its universal distress signal: **SOS (`··· ––– ···`)**.

---

## 📜 Historical Context
Invented in the 1830s by American artist and inventor **Samuel F. B. Morse** and his assistant **Alfred Vail**, Morse code was developed specifically to pair with the electric telegraph. This breakthrough technology allowed messages to be transmitted across vast distances almost instantly. 

* **The First Message:** In 1844, the first official telegraph message—*"What hath God wrought?"*—was successfully transmitted from Washington, D.C., to Baltimore.
* **Global Standardization:** As telegraph networks expanded worldwide, it became the backbone of international business, journalism, and military operations. By the mid-19th century, European nations refined the system into **International Morse Code**, which remains the global standard today.

---

## ⚙️ How It Works

The system relies on a precise combination of signals and timing:

* **Dots (·):** Short, rapid signals.
* **Dashes (–):** Longer signals (equivalent to three dots in duration).
* **Spacing:** * Short pauses separate individual dots and dashes within a letter.
  * Medium pauses separate distinct letters.
  * Long pauses separate words.

### ⚡ Design Efficiency
Morse code is inherently optimized for speed. Frequently used letters in the English language are assigned the shortest sequences, while rare letters receive longer ones:

| Character | Morse Code | Length |
| :---: | :---: | :---: |
| **E** | `·` | Shortest (Most Common) |
| **T** | `–` | Short Short |
| **A** | `· –` | Medium |
| **B** | `– · · ·` | Long |
| **SOS** | `··· ––– ···` | Universal Distress Signal |

This brilliant design choice made telegraph communication significantly faster, more practical, and less prone to transmission errors.
