## Morse code
#Overview

Morse code is a communication system that represents letters, numbers, and punctuation using sequences of dots (·) and dashes (–). These signals can be transmitted through sound, light, or electrical pulses, making it one of the earliest forms of long-distance digital communication. It is most famously known for the distress signal SOS (··· ––– ···).

#The historical context of why Morse code was created.
Morse code was invented in the 1830s by Samuel F. B. Morse, an American artist and inventor, along with his assistant Alfred Vail. It was designed to work with the electric telegraph, enabling messages to be sent quickly across long distances. The first official message, “What hath God wrought?”, was transmitted in 1844 from Washington, D.C. to Baltimore.
As telegraph networks expanded, Morse code became the backbone of international communication, playing a vital role in business, journalism, and military operations. By the mid-19th century, European nations developed the International Morse Code, which became the global standard.

#An explanation of how the Morse code system works, including the use of dots and
dashes.
Each character (letter, number, or symbol) is represented by a unique sequence of dots and dashes.

Dots are short signals, while dashes are longer signals.

Spaces between dots and dashes distinguish characters, while longer pauses separate words.

Example:

A = · –

B = – · · ·

SOS = · · · – – – · · ·

Morse code’s efficiency lies in its simplicity: frequently used letters (like E and T) have shorter codes, while less common letters have longer ones. This design made telegraph communication faster and more practical.
